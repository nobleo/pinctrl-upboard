# pinctrl-upboard
UP boards pin controller driver for UP HAT pins.

Supported products
------------------------------------------------
* UP Squared/UP Squared Pro
* UP Core
* UP Core Plus
* UP Xtreme
* UP Xtreme i11
* UP Xtreme i12
* UP Squared 6000
* UP Element
